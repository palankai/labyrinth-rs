# Simple labyrinth-based game (WIP)

## How to run

It is tested on Apple M1 MacOS Ventura.
However, it probably works on Linux too, maybe even on Windows.

```
cargo run
```

### Controls
- Menu: P -> Play
- Menu: Q -> Quit

- Game: WASD / Up/Left/Down/Right Controls the player
- Game: ESC -> Back to menu
